## 教材说明

- **语文**：词库依据课本「写字表」逐课导入（22 课 + 8 个语文园地），包含 2025 春新版《热爱中国共产党》《浪花》。
- **英语**：2025 新版 6 Units 扁平化结构（Food / Classroom / Activities / Seasons / Fruit / Animals），每单元配句型模板。
- 所有词库放在 HTML 顶部 `const UNITS`，**家长可对照纸质课本随时修订**。

## 使用方法

1. 访问 <https://tangjianfang.github.io/worksheets/>，或直接下载对应 HTML 文件用浏览器打开
2. 选择单元 / 范围，勾选所需题型（修改后自动重新出题）
3. 点击「⟳ 重新生成」可随机换一套题
4. 点击「🖨 打印」或 `Ctrl+P`，A4 纸直接打印
5. 每个练习页左上角「← 首页」按钮可返回主导航

> **微信内置浏览器**：点击打印后会弹出提示，按提示先在外部浏览器打开再打印。
