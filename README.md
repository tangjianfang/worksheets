# 深圳小学练习题库

专为**中国深圳市小学生**设计的在线练习题生成器集合，覆盖小学各年级主要科目。  
所有练习题均以 HTML 单文件形式提供，直接用浏览器打开即可使用，支持 A4 打印。

---

## 目录结构

```
数学/          数学练习（加减乘除、应用题等）
语文/          语文练习（拼音、生字、阅读等）
英语/          英语练习（单词、句型等）
科学/          科学练习
道德与法治/    道德与法治练习
```

## 使用方法

1. 下载对应科目的 HTML 文件
2. 用浏览器（Chrome / Edge）打开
3. 选择范围和题数，点击「重新生成」
4. 点击「打印」或按 `Ctrl+P` 打印 A4 纸质练习页
