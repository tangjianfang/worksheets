# 深圳小学练习题库

专为**中国深圳市小学生**设计的在线练习题生成器集合，覆盖小学各年级主要科目。  
所有练习题均以 HTML 单文件形式提供，直接用浏览器打开即可使用，支持 A4 打印。

---

## 目录结构

```
数学/          数学练习（加减乘除、应用题等）
语文/          语文练习（拼音、生字、阅读等）
英语/          英语练习（单词、句型等）
科学/          科学练习
道德与法治/    道德与法治练习
```

### 教材版本说明

- **语文**：人教社统编版（部编版）一年级下册。词库参照 2017 版「写字表」整理，2025 春新版新增的《热爱中国共产党》《浪花》两课暂未收录。
- **英语**：沪教牛津深圳版 1B，旧版 4 Modules × 3 Units = 12 Units。2024 审定的新版（6 Units 扁平化结构）暂未收录。
- 所有词库与拼音均放在 HTML 文件顶部 `const UNITS` 中，**家长可按家中纸质课本逐项核对/修订**。

## 使用方法

1. 下载对应科目的 HTML 文件
2. 用浏览器（Chrome / Edge）打开
3. 选择范围和题数，点击「重新生成」
4. 点击「打印」或按 `Ctrl+P` 打印 A4 纸质练习页
